Privacy & Security

Who Should Have Access

This inventory system is intended for Harvey Norman staff only. That includes store managers, department supervisors and stock team members. No one outside of the store team should be able to view or change inventory data. Customers, contractors or members of the public should never have access to this system.

How Access Should Be Controlled

In a real deployment of this system there are two main ways to make sure only the right people can get in.

The first option is a login system. Staff would need a username and password to access the app. Managers could have extra permissions like deleting items, while regular staff might only be able to view or add stock. Passwords should be strong and sessions should time out after a period of inactivity so a logged in computer left unattended cannot be misused.

The second option is an IP whitelist. This means the system would only load if the request is coming from an approved IP address, such as the store network. Anyone trying to access it from outside the store network, like from home or on mobile data, would be blocked automatically. This works well for an in-store system where staff only need access while they are at work.

For the best security both options could be used together. Staff would need to be on the store network and also logged in before they can access the inventory.

Input Validation

This app checks all form inputs before saving anything. Fields like Quantity and Price only accept numbers and all required fields must be filled. This stops bad data from getting into the system and also protects against basic attacks where someone tries to enter harmful content into a text field.

Data and Privacy

This version of the app stores data in memory only. When the browser is closed or refreshed all data is gone. In a real system the inventory data would be stored in a database. That database should be secured so that only the application can talk to it and the data should be backed up regularly in case something goes wrong.

Australian businesses also need to follow the Privacy Act 1988. Even though inventory data is not personal information, supplier details and pricing information is sensitive and should not be shared outside the business.
