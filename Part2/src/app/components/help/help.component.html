Quick Start

  • Go to Inventory to add, edit or delete items
  • To edit — type the item name in Load Item to Edit, click Load, make changes, click Update Item
  • To delete — type the item name in Delete Item, click Delete, confirm the prompt
  • Go to Search to find items by name or filter by category and stock status
  • Use the Popular Items button on the Inventory page to view top products
